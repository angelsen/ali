# ALI Roadmap

## Version 2.0 (Completed)

### Core Features
- [x] Template engine with conditionals and array lookups
- [x] Service composition system
- [x] Cross-plugin grammar resolution
- [x] Selector framework (stream/action)
- [x] Multi-plugin verb support
- [x] Plugin integration (`--init`)
- [x] Structured logging with environment capture

### Plugin Features
- [x] Popup support (`pop` direction)
- [x] COPY/ECHO verbs for stream operations
- [x] Spatial-aware pane distribution
- [x] Plugin best practices documentation

## Version 2.1 (Next)

### Enhanced Selectors
- [ ] Multi-stage selectors (e.g., `br? | ed?`)
- [ ] Conditional selectors based on context
- [ ] Custom selector pipelines

### Plugin Ecosystem
- [ ] Git plugin for common operations
- [ ] Docker plugin for container management
- [ ] SSH plugin for remote operations
- [ ] Package manager integration (npm, cargo, etc.)

### Template Enhancements
- [ ] Nested template resolution
- [ ] Template functions (uppercase, lowercase, etc.)
- [ ] Template loops for batch operations

## Version 3.0 (Future)

### Advanced Features
- [ ] Plugin marketplace/registry
- [ ] Plugin dependency resolution
- [ ] Command history and replay
- [ ] Interactive mode improvements
- [ ] Shell completion scripts

### Performance
- [ ] Compiled template cache
- [ ] Lazy plugin loading
- [ ] Parallel command execution support

## Long-term Vision

- **Plugin ecosystem** - Rich library of community plugins
- **Universal interface** - Single syntax for all CLI tools
- **Composability** - Complex workflows from simple commands
- **Learning system** - Adapt to user patterns
- **Cross-platform** - Full Windows and macOS support
